# Compact Syntax Document for Markdown-like Format

## Structural Tags
- `title=Example Title`

## Heading Tags
- `# Heading 1` (`<h1>`)
- `## Heading 2` (`<h2>`)
- `### Heading 3` (`<h3>`)
- `#### Heading 4` (`<h4>`)
- `##### Heading 5` (`<h5>`)
- `###### Heading 6` (`<h6>`)

## Text Formatting Tags
- Paragraph: `This is a paragraph.` (`<p>`)
- Strong: `**Strong Text**` (`<strong>`)
- Emphasis: `*Emphasized Text*` (`<em>`)
- Bold: `**Bold Text**` (`<b>`)
- Italics: `*Italic Text*` (`<i>`)
- Highlight: `==Highlighted Text==` (`<mark>`)
- Blockquote: 
  `> This is a blockquote.` (`<blockquote>`)
- Preformatted: 
  ```
  preformatted text
  ``` (`<pre>`)

## List Tags
- Unordered List:
  ```
  - Item 1
    - Subitem 1
  ```
  (`<ul>`, `<li>`)
  
- Ordered List:
  ```
  1. First Item
     1. Subitem
  ```
  (`<ol>`, `<li>`)

## Link and Anchor Tags
- Link: `[Link Text](http://example.com)` (`<a>`)

## Image Tags
- Image: `![Alt Text](image_url)` (`<img>`)

## Table Tags
```
| Header 1 | Header 2 |
|----------|----------|
| Row 1    | Data 1   |
| Row 2    | Data 2   |
```
(`table`, `tr`, `th`, `td`)

## Form Tags
- Input: `input[placeholder](type){label}`
- Textarea: `input[placeholder](textarea){label}`
- Button: `button[Click Me]{label}`
- Select: `select["Option 1", "Option 2"]{label}`
  
## Semantic Tags
- Header: 
  ```
  header
    # Header Content
  ```
  (`<header>`)
  
- Footer: 
  ```
  footer
    # Footer Content
  ```
  (`<footer>`)
  
- Navigation: 
  ```
  nav
    - [Home](http://example.com)
    - [About](http://example.com/about)
  ```
  (`<nav>`)
  
- Article: 
  ```
  article
    # Article Title
    This is an article.
  ```
  (`<article>`)
  
- Section: 
  ```
  section
    # Section Title
    This is a section.
  ```
  (`<section>`)
  
- Aside: 
  ```
  aside
    # Aside Content
  ```
  (`<aside>`)
  
- Main: 
  ```
  main
    # Main Content
  ```
  (`<main>`)
